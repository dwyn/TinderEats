## Installation 

1. run:
```ruby

```

```ruby

```

3. run:
```ruby
gem install bundler
```

4. run:
```ruby
gem install pry
```

5. run:
```sh
brew install postgres
```

6. run:
```sh
brew services start postgresql
```

7. run:
```sh
gem install pg
```
